# WFM API
### Simple Workforce Management API built using .NET 9

#### Application details:

    ✅ API design using ASP.NET Core Web API.
  
    ✅ Used PostgreSQL as the relational database.
  
    ✅ Used Entity Framework Core for ORM and database migrations.
  
    ✅ API documentation and testing with Swagger UI.
  
    ✅ Error handling and validation.
  
    ✅ Authentication and authorization implementing 2 roles: Admin and Regular user.
  
    ✅ Three-layer architecture.

    ✅ Supports CRUD operations for employees in a company.

The API will be consumed by an Angular application, which is currently in progress.

Angular app link: https://github.com/danaildimovv/wfm-ui
